## Easy

<pre><span style="font-size:18px"><strong>Input: </strong>N = 7, K = 2
arr[] = {1, 2, 3, 4, 5, 6, 7}
<strong>Output:</strong> 3 4 5 6 7 1 2
<strong>Explanation</strong>: Rotation of the above 
array by 2 will make the output array .</span></pre>
